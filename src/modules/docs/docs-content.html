Code Standards

We adhere to strict SOLID principles. Every function must have a single responsibility, and every component must be purely semantic.

example-service.ts
async function calculateCompetency(employeeId: string): Promise {
  const employee = await repository.getById(employeeId);
  const scores = employee.skills.map(s => s.score);
  return scores.reduce((a, b) => a + b, 0) / scores.length;
}
